## ü§ù Sharing & Collaboration Features

### **Share Button Location:**
1. **Open Draft Manager**
   - Click on your draft name in the header (top left)
   - OR click "Manage All" in the draft dropdown
   
2. **Find Share Button**
   - Look for the **Share icon (üì§)** next to each draft

   - Green color on hover
   
3. **Share Modal**
   - Click Share button ‚Üí Opens Share Project modal
   - Enter email, select permission, click "Send Invite"

### **Invitation Notifications:**
- **Location**: Top right of header
- **Icon**: Bell icon (üîî) with red badge showing count
- **When visible**: Only shows when you have pending invitations
- **Click**: Opens invitation list to accept/decline

## ü§ñ AI Features

### **1. Executive Summary AI Assistant**
- **Location**: Executive Summary tab
- **Where**: Scroll down past the form fields
- **Look for**: Blue gradient box with "AI Writing Assistant" heading
- **Icon**: Sparkles icon (‚ú®)
- **How to use**: Type a question and click "Get AI Assistance"

### **2. Market Analysis AI**
- **Location**: Market Analysis tab
- **Where**: Click "Location Insights" sub-tab
- **Features**:

  - **AI Assistant** - Scroll down, blue box with "AI Assistant" heading
- **How to use**: Ask questions about market trends, demographics, competition

### **3. Financial Projections AI Advisor**
- **Location**: Financial Projections tab
- **Where**: Scroll down to "AI Financial Advisor" section
- **Look for**: Indigo/purple section card with "AI Financial Advisor" title
- **Icon**: Sparkles icon (‚ú®)
- **How to use**: Ask questions like "What are the biggest risks in my financial plan?"

## üìÑ Document Import Feature

### **Location**: 
- **Tab**: "Import Document" (first tab in navigation)
- **Icon**: FileText icon
- **Description**: "Upload & extract data"

### **How to Use**:
1. Click "Import Document" tab
2. Click "Choose File" button
3. Select PDF, Word, or text file
4. Wait for AI to extract data
5. Select sections to import
6. Click "Import Selected Sections"

## üîç Troubleshooting

### **Can't find Share button?**
- Make sure you're in Draft Manager (click draft name in header)
- Look for the green Share icon (üì§) - it's the first action button
- Make sure you have at least one draft created

### **Can't find AI features?**
- **Executive Summary**: Scroll down past the form - it's at the bottom
- **Market Analysis**: Click "Location Insights" tab first
- **Financial Projections**: Scroll down - it's near the bottom, after all the calculators

### **AI not responding?**
- Check your `.env.local` file has `REACT_APP_OPENAI_API_KEY` set
- Make sure API key is valid
- Check browser console for errors

### **Share button not working?**
- Make sure you're logged in
- Check browser console for errors
- Verify Firestore rules are deployed
