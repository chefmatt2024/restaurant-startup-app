# Consulting Lead Tracker — Action 3

Use this (or copy to a spreadsheet) to track consulting leads until you have a CRM.

---

## How to use

1. **New lead** → Add a row with date, source, name/email, interest, status
2. **Weekly review** → Move leads through stages: New → Contacted → Proposal sent → Won / Lost
3. **Source tagging** → Use UTM: assessment, dashboard, landing, iterumfoods.com, linkedin, facebook, etc.

---

## Sources to tag (from UTM)

- `utm_source=assessment` — Assessment results "Get a 30-min review"
- `utm_source=dashboard` — Dashboard "Book a strategy call"
- `utm_source=landing` — Landing page "Book a Strategy Call"
- `utm_source=iterumfoods` — iterumfoods.xyz or iterumfoods.com
- `utm_source=linkedin` — LinkedIn
- `utm_source=facebook` — Facebook
- `utm_source=instagram` — Instagram
- `utm_source=medium` — Medium post
